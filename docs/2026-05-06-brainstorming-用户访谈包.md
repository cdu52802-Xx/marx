## 访谈包 ① · 1 页产品概念描述（直接 copy 给目标用户看）

> **Marx 星图**——一个交互式可视化产品，把 Marx 一生中遇到的人（恩格斯、青年黑格尔派、第一国际同志、主要论敌如蒲鲁东 / 巴枯宁）、他写的书（《共产党宣言》《资本论》《1844 年经济学哲学手稿》等）、他参与的事件（1848 革命、巴黎公社、第一国际）、以及他思想里的关键概念（异化 / 剩余价值 / 历史唯物主义 / 阶级斗争……）做成一张以 Marx 为中心的关系星图。
>
> 拖动时间轴 / 切换地理足迹视图，可以看见这些「人 / 书 / 事件 / 概念」如何在 1818-1883 年这 65 年间一步步浮现、相互作用、并塑造出 Marx 的思想。每个节点都带学术引用源（MIA / SEP / MEGA / Wikidata），中文为主，关键术语保留德文/英文原文。
>
> **它解决什么**：现有的 Marx 学习资源——SEP / 维基 / 通俗读物——把人、思想、时代分开讲。读者很难「看见」思想的形成路径。这个产品把所有维度合一，让「参透 Marx 思想形成」变成一次可视化探索。
>
> **适合谁**：哲学/政治经济学/社科 研究者、进阶读者、想真正理解 Marx 思想形成路径的人。

## 访谈包 ② · 访谈话题（5-7 题，30 分钟）

1. **场景**：你了解 Marx 时，最常被卡住的地方是什么？
2. **现状**：你目前怎么解决这些卡顿——读书 / 维基 / SEP / B 站 / 别的？
3. **概念测试**（这时候给他们看上面的 1 页概念描述）：这个东西看起来对你有用吗？哪个场景下你会真的打开来用？
4. **差异感**：跟你已经在用的（Marxists.org / SEP / 自己的笔记）相比，这个独特吸引点在哪里？
5. **反向问**（**这题最值钱**）：哪些功能你会觉得「看起来酷但其实不会用」？
6. **付费意愿**：免费 / 一次性付 30 元 / 订阅 10 元/月，你最可能怎么选？为什么？
7. **邀请测试**：原型出来后愿意当第一批试用者吗？
